# **App Name**: PDFWrite

## Core Features:

- PDF Upload & Parsing: Allow users to upload PDF files (image-based or direct-text) and parse their content.
- Intelligent Text Extraction: Use a tool powered by generative AI to intelligently extract text, preserving the original PDF structure (headings, paragraphs, lists).
- Content Formatting: Automatically format the extracted content into a structured text format similar to Notion's import style (headings, bullet points, paragraphs).
- Text Editor: Provide a text editor so users can directly adjust content
- Download Formatted Text: Allow users to download the formatted text as a .txt file.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to convey reliability and innovation.
- Background color: A light blue (#E1F5FE), very slightly desaturated from the primary, for a clean and calming interface.
- Accent color: A complementary purple (#9C27B0), for secondary CTAs and highlights.
- Body text: 'Inter', a grotesque sans-serif font, for a modern and neutral reading experience.
- Headline text: 'Space Grotesk', a proportional sans-serif font, to match 'Inter'.
- Use clear and concise icons for actions like upload, download, and editing.
- A clean, intuitive layout with a focus on readability. Prioritize a distraction-free text editing environment.
- Subtle animations to provide feedback during the PDF parsing and text formatting processes.
